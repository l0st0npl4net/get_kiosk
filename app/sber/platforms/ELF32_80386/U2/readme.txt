Для установки агента обновления u2u1 под Linux необходимо:
1. Скопировать на кассу в каталог с файлами UPOS-1 все файлы из каталога U2.

2. Если пинпад подключается по USB/RS232 необходимо:

   2.1. В файле config.ini установить параметр AgentURL=127.0.0.1:8000
   2.2. В файле pinpad.ini в параметре ComPort указать номер СОМ-порта (для USB указать 9).
   
   2.3. Настроить автоматический запуск файла launcher при старте системы. Для этого нужно:
          -войти в систему под аккаунтом, под которым будет работать кассовая программа;
          -дать команду crontab -e
          -в открывшемся текстовом редакторе набрать строку:

           @reboot cd <путь к каталогу с файлами UPOS 2.0>;./launcher

          -сохранить файл и выйти из редактора.

3. Если пинпад подключается напрямую к Ethernet, то настраивать запуск программы launcher не требуется. 
   Вместо этого следует в файле config.ini установить параметр AgentURL=X.X.X.X:8000, где X.X.X.X - фиксированный IP-адрес пинпада.
 
4. Убедиться, что адрес 185.157.96.41:80 доступен с кассы.

5. Выполнить сверку итогов для проверки работоспособности. При этом в каталоге должны появиться подкаталоги Update. Сверка итогов должна пройти успешно, без ощутимых задержек.
